# Стили (SCSS)

## Структура
```
styles/
    ├── global.scss        # Глобальные стили. (Сброс стилей, базовые стили для html / body)
    ├── _variables.scss    # Переменные
    ├── _fonts.scss        # Подключение шрифтов
    ├── mixins/            
        ├── _mixins-block.scss        # Миксины для блоков    
        ├── _mixins-font.scss         # Миксины для шрифтов          
        └── _mixins-utils.scss        # Миксины утилиты              
    └── functions/
        └── _functions-utils.scss     # Функции утилиты    
```
